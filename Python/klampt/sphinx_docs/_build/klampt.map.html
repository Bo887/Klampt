klampt.map module¶

Helpers for accessing world variables using a class-style interface

You can, for example, write map(world).robots[0].config to retrieve the name of robot 0, rather than world.robot(0).getConfig().

You can also write map(world).robots[0].config = [q1,...,qn] to set the configuration of robot 0, rather than world.robot(0).getConfig([q1,...,qn]).

Conveniently, you can write expressions like map(world).robots[0].config[4] = 3.5
