klampt.rootfind.findRoots(PyObject * startVals, int iter) → PyObject *[source]¶

PyObject* findRoots(PyObject *startVals, int iter)

Performs unconstrained root finding for up to iter iterations Return values is a tuple indicating: (0,x,n) : convergence reached in x

(1,x,n) : convergence reached in f

(2,x,n) : divergence

(3,x,n) : degeneration of gradient (local extremum or saddle point)

(4,x,n) : maximum iterations reached

(5,x,n) : numerical error occurred where x is the final point and n is the number of iterations used
