klampt.se3 module¶

Operations for rigid transformations in Klamp’t. All rigid transformations are specified in the form (R,T), where R is a 9-list specifying the entries of the rotation matrix in column major form, and t is a 3-list specifying the translation vector.

These are useful for interfacing with C code.

klampt.se3.apply(T, point)[source]¶

Applies the transform T to the given point

klampt.se3.apply_rotation(T, point)[source]¶

Applies only the rotation part of T

klampt.se3.distance(T1, T2, Rweight=1.0, tweight=1.0)[source]¶

Returns a distance metric between the two transformations. The rotation distance is weighted by Rweight and the translation distance is weighted by tweight

klampt.se3.error(T1, T2)[source]¶

Returns a 6D “difference vector” that describes how far T1 is from T2. More precisely, this is the Lie derivative (w,v).

klampt.se3.from_homogeneous(mat)[source]¶

Returns a T corresponding to the 4x4 homogeneous transform mat

klampt.se3.from_rotation(mat)[source]¶

Returns a T corresponding to the 3x3 rotation matrix mat

klampt.se3.homogeneous(T)[source]¶

Returns the 4x4 homogeneous transform corresponding to T

klampt.se3.identity()[source]¶

Returns the identity transformation.

klampt.se3.interpolate(T1, T2, u)[source]¶

Interpolate linearly between the two transformations T1 and T2.

klampt.se3.inv(T)[source]¶

Returns the inverse of the transformation.

klampt.se3.mul(T1, T2)[source]¶

Composes two transformations.

klampt.se3.rotation(T)[source]¶

Returns the 3x3 rotation matrix corresponding to T’s rotation
